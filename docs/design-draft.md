# 喵星复兴计划 · 产品简介

> **文档版本：** v2.0
> **最后更新：** 2026-06-22

---

## 一、产品定位

一款 Web 端番茄钟 + 等距城建游戏。用户通过番茄钟完成专注任务获取燃料，逐步将喵星从一个小纸箱建设成繁华都市，过程中有各种小猫咪的陪伴与互动。

- **核心用户**：15–30 岁需要自我管理能力的学生与初入职场年轻人
- **使用场景**：学习备考、完成作业、阅读打卡、工作专注、习惯养成
- **解决痛点**：传统效率工具枯燥易弃、单纯游戏带来负罪感；本作品将「专注力培养」与「猫咪陪伴 + 城市建造」结合，让自律有温度

---

## 二、核心玩法

| 维度 | 设计要点 |
| --- | --- |
| 输入 | 开启一个 25 分钟番茄钟专注时段 |
| 产出 | 专注完成后获得「燃料」与经验值（XP） |
| 反馈 | 经验值累积 → 等级提升 → 解锁迷雾区域 / 新建筑 |
| 陪伴 | 地图上随机出现可互动的小猫咪，附带对话气泡与故事 |
| 长期目标 | 将喵星从「一个纸箱」逐步建设为「繁华都市」 |

### 关键交互

1. **番茄钟**：25 分钟倒计时，含开始 / 暂停 / 重置，配合猫表情反馈
2. **等距地图（Isometric Map）**：基于网格的伪 3D 瓦片地图，支持缩放与拖拽平移
3. **迷雾系统**：未解锁区域以迷雾覆盖，完成番茄钟逐步揭开
4. **建筑系统**：2.5D 风格建筑，通过 XP 解锁，按成长阶段递进（荒野 → 都市）
5. **猫咪角色**：地图上独立活动的猫，带有对话气泡和随机互动
6. **等级与进度**：XP 条与等级可视化，反馈长期成长

## 三、视觉与体验方向

- **整体风格**：深色治愈系 + 等距 2.5D 半扁平 low-poly
- **主色调**：
  - 背景深蓝：`#0a0e1a`
  - 渐变装饰光晕（紫蓝 / 青蓝）
  - 强调色：番茄橙 `#FF6B35`
- **氛围关键词**：治愈、陪伴、成就感、缓慢成长
- **动效**：GSAP 滚动触发动画、视差、光标跟随，弱化机械感、强化情感连接
- **响应式**：PC + 移动端双端适配（断点 768px）
- **美术实现**：全部程序化代码生成（`src/game/render/`），无图片依赖
