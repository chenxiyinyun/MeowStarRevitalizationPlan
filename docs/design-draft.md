## 二、项目标题

> **【生活娱乐赛道】喵星复兴计划——用番茄钟养猫建城，让每一次专注都有意义**

---

## 三、创意名称与创意介绍

### 创意名称
喵星复兴计划

### 想解决什么问题
许多青少年和年轻人在学习、工作中难以保持专注。传统的效率工具枯燥乏味，缺乏持续使用的动力，导致计划半途而废。

### 为什么会想到做这个
番茄钟等效率工具虽然有效，但用户很难坚持；而猫咪和建造类游戏天然具有治愈感和成就感。将「专注力培养」与「猫咪陪伴 + 城市建造」结合，让自律变成一件有温度的事。

### 大概是什么产品
一款兼容 PC 和移动端的 Web 应用。用户通过番茄钟完成任务获取燃料，逐步将喵星从一个小纸箱建设成繁华都市，过程中还有各种小猫咪的待机互动和趣味故事发生。

---

## 四、目标用户及痛点

### 面向哪些用户
核心用户为 **15–30 岁** 的青少年及年轻群体，尤其是需要自我管理能力但容易分心、缺乏动力的学生党和初入职场的年轻人。

### 在什么场景下使用
- 学习备考
- 完成作业
- 阅读打卡
- 工作专注
- 习惯养成等需要定时专注的日常场景

### 当前痛点
- 现有番茄钟 / 效率工具：界面单调、反馈感弱，用户往往用几天就放弃。
- 单纯的游戏：虽然有趣，却会带来「浪费时间」的负罪感。
- **用户缺少一个既能提升专注力、又能获得正向情绪反馈的工具。**

---

## 五、价值与意义

### 社会价值
- 关注青少年身心健康，通过游戏化的正向激励机制帮助用户建立良好的时间管理习惯。
- 减少因拖延和效率低下带来的焦虑感。
- 治愈系的猫咪互动和可视化的成长反馈，能在心理层面提供陪伴感和成就感，对抗孤独与内耗。

### 商业价值
- 效率工具 + 休闲游戏的跨界融合具有差异化竞争力。
- 猫咪 IP 和城市建造体系为后续拓展周边、主题皮肤、社交分享等商业化路径提供了丰富空间。

## 六、核心玩法（基于原型网站提炼）

| 维度 | 设计要点 |
| --- | --- |
| 输入 | 开启一个 25 分钟番茄钟专注时段 |
| 产出 | 专注完成后获得「燃料」与经验值（XP） |
| 反馈 | 经验值累积 → 等级提升 → 解锁迷雾区域 / 新建筑 |
| 陪伴 | 地图上随机出现可互动的小猫咪，附带对话气泡与故事 |
| 长期目标 | 将喵星从「一个纸箱」逐步建设为「繁华都市」 |

### 关键交互
1. **番茄钟**：25 分钟倒计时，含开始 / 暂停 / 重置，配合猫表情反馈。
2. **等距地图（Isometric Map）**：基于网格的伪 3D 瓦片地图，支持缩放与拖拽平移。
3. **迷雾系统**：未解锁区域以迷雾覆盖，完成番茄钟逐步揭开。
4. **建筑系统**：2.5D 风格建筑（草地、泥土、森林、水面），通过 XP 解锁。
5. **猫咪角色**：地图上独立活动的猫，带有对话气泡和随机互动。
6. **等级与进度**：XP 条与等级可视化，反馈长期成长。

## 七、视觉与体验方向

- **整体风格：** 深色治愈系 + 等距 2.5D 像素 / 半扁平风
- **主色调：**
  - 背景深蓝：`#0a0e1a`
  - 渐变装饰光晕（紫蓝 / 青蓝）
- **氛围关键词：** 治愈、陪伴、成就感、缓慢成长
- **动效：** GSAP 滚动触发动画、视差、光标跟随，弱化机械感、强化情感连接
- **响应式：** PC + 移动端双端适配（断点 768px）
